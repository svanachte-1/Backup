Les Voitures

Elles filent à toute allure,Les voitures, les voitures.Au fond de villes obscure Comme sous cieux d'azur!

Belle mécanique montures, Les voitures, les voitures. Sont la bête signature De leurs maîtres qui assurent.

Elles filent à toute allure, Les voitures, les voitures. Mais polluent notre air pur Et font vilains murmures !

Quand vient leur point d'usure, Les voitures, les voitures, N'ont plus très fière allure et finissent en ordure...
